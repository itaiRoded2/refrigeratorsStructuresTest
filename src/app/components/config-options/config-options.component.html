This structure here is scalable and we add add modify to fit our old, current and future fridges.
In our code we demo creating 2 different types of fridges (we can of course create as many we need for testing) For each we call findMyFood on 3 different items: milk, gouda, veggies

We will notice the same call, to find the same item will return (as expected) different results based on the fridge configuration
